About us

Fydor

Fydor is a local-first language learning app for turning lesson material into structured study. It helps learners create or import lessons, study sentence-rich content, review what they forget, and share portable lesson packs.

What it is

A calm study space for lessons that belong to the learner.

Build and import lessons

Create structured lessons or bring in existing material with translations, notes, vocabulary, and example sentences.

Practice in context

Study the same saved sentences through flashcards, fill-in-the-blank prompts, multiple choice, and recall review.

Review what matters

Fydor uses a lightweight spaced repetition queue so missed sentences return sooner and familiar ones wait longer.

Open source

The app source lives on GitHub.

The public application repository contains the Next.js app, Tauri desktop shell, local persistence layer, lesson import tools, study modes, and review system behind Fydor.

Why it exists

Useful language practice should stay portable.

Fydor is built for students, teachers, and self-learners who want reusable language materials without subscriptions, paywalls, or a hosted account. Lesson packs can move between machines, and study data stays local to the app.
