Mac install

If your Mac blocks Fydor on first launch.

This download is currently unsigned and unnotarized, so macOS may show a warning on first launch. That is expected for this build.

Best path

Install the current build so you can drag Fydor into Applications and open it normally once macOS allows it.

If macOS warns

If you see a warning, open System Settings > Privacy & Security and click Open Anyway, or Control-click Fydor in Finder and choose Open.

Last resort

If Fydor still won’t open, you can clear the quarantine flag with xattr -dr com.apple.quarantine "/Applications/Fydor.app".
