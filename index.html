Mac install

If macOS blocks the app on first launch.

Best path

Ship a signed and notarized build so most users can just drag Fydor into Applications and open it normally.

If macOS warns

Open System Settings > Privacy & Security and click Open Anyway, or Control-click the app in Finder and choose Open.

Last resort

For troubleshooting only, some users clear the quarantine flag with xattr -dr com.apple.quarantine "/Applications/Fydor.app".
